# Data Description

## ratings.dat
- Original data before preprocessing
- uid \t iid

## u.data
- Preprocessed data before splitting
- format: uid \t iid

## LOOTrain.dat, LOOVal.dat, LOOTest.dat
- Training data, Validation data, Test data
- format: uid \t iid

## LOONegatives.dat
- uid \t neg1 \t neg2 \t...\t neg100
